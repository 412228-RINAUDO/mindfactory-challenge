# mindfactory-challenge

Proyecto con frontend (React + Vite) y backend (Express + TypeScript)

## Estructura

```
├── frontend/    # React + Vite
└── backend/     # Express + TypeScript
```

## Instalación

```bash
# Frontend
cd frontend
pnpm install

# Backend
cd backend
pnpm install
```

## Desarrollo

```bash
# Frontend (puerto 3000)
cd frontend
pnpm dev

# Backend (puerto 4000)
cd backend
pnpm dev
```

## Build

```bash
# Frontend
cd frontend
pnpm build

# Backend
cd backend
pnpm build
```
